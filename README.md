# ParkFlow - Parking Management System 🚗

ParkFlow is a web-based parking management system designed to automate vehicle entry, exit, slot allocation, and billing. The system provides real-time parking slot monitoring and maintains transaction records for efficient parking operations.

## Features

* Secure login system for Admin and Attendant
* Automatic parking slot allocation
* Vehicle entry and exit management
* Real-time parking slot availability tracking
* Automated parking fee calculation
* Invoice generation and printing
* Transaction history and revenue reports
* Interactive dashboard with parking statistics

## How to Run

1. Clone the repository.
2. Import the MySQL database using `Schema.sql`.
3. Install required Python packages:

```bash
pip install flask flask-cors mysql-connector-python
```

4. Start the Flask server:

```bash
python app.py
```

5. Open `index.html` in your browser.
